How should the admin login work?

Only you should be able to access /admin — pick how you log in

A

Single Secret Password Recommended

One strong password set in your environment variables. Visit /admin, enter the password, get a session cookie. Simple, no accounts needed, impossible to guess if the password is strong.

✓ Zero setup ✓ Works immediately ✓ Change password by updating env var
B

Email + Password (Supabase Auth)

Log in with your email and a password through Supabase's built-in auth system. Supports multiple admin users, password reset by email, and proper JWT sessions.

✓ Multiple admins possible ✓ Password reset via email ⚠ Slightly more setup
